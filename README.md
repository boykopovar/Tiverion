## 🚀 Кратко

**Tiverion** — ASP.NET Core (C#) веб-приложение для сбора погодных данных по часам (из одной или нескольких точек на карте) и статистического/вероятностного анализа событий: биномиальные оценки, геометрическое распределение, условные вероятности между индикаторами (I₁…I₄) и пользовательскими порогами. Предназначено как инструмент для метеорологов и исследователей.

---

## 🎯 Миссия

Предоставить удобный, наглядный и расширяемый инструмент для выявления вероятностных связей между параметрами погоды и событием (дождь, буря, усиление ветра и т.п.), чтобы:

* быстро тестировать гипотезы (P(A|B)),
* получать оценку частотности (биномиальная модель),
* оценивать «время до события» (геометрическое распределение),
* экспортировать отчёты и строить уведомления.

---

## ✨ Возможности

* Выбор точки(ок) на интерактивной карте (или импорты координат).
* Конфигурируемые индикаторы и пороги (например, падение давления ≥5 мм/6ч, ветер ≥12 м/с, Kp ≥5).
* Подсчёт: биномиальные вероятности, доверительные интервалы, частоты событий.
* Анализ геометрического распределения (сколько часов/дней до первого события).
* Расчёт условных вероятностей P(I₂|I₁), P(I₂|I₃), P(I₄|I₁) и т. п.
* Визуализация: гистограммы, кумулятивные графики, тепловые матрицы корреляций и временные сериалы.
* Уведомления (вебхук / email) при нахождении статистически значимой взаимосвязи.
* Экспорт результатов в CSV / JSON / PDF / EXCEL.
